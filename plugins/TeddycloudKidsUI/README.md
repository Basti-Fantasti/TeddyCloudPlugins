# TeddyCloud Kids UI Plugin

A kid-friendly interface for assigning audio content to Tonie tags. Designed for tablets and touch devices, with large buttons, simple navigation, and a guided flow.

---

## Features

- Splash screen with a single large start button
- Automatic single-box detection, or selection grid for multiple boxes
- Tag placement screen with live polling
- Audio library with search and pagination
- Confirmation step before linking
- Multi-language interface (German / English)
- Theme adaptation from the parent TeddyCloud UI (light / dark)
- Fullscreen toggle for immersive use on tablets

---

## Settings (since 0.3.0)

The settings dialog (gear icon on the splash screen) lets you change items per page, list and detail icon sizes, and the interface language. Choices are stored in the browser's `localStorage`, so they stay on the device they were saved on.

---

## Installation

1. Copy this folder (`TeddycloudKidsUI`) into the TeddyCloud plugin directory
2. Restart TeddyCloud or reload plugins
3. The plugin appears in the plugin list
